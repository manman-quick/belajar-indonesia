# Belajar Indonesia - 设计文档

## 品牌定位
一款面向中文母语者的印尼语学习应用，以热带岛屿文化为视觉灵感，融合现代扁平化设计与游戏化学习体验。

## 配色方案
- **主色（Primary）**：珊瑚橙 `#FF6B35`（热带活力）
- **辅色（Secondary）**：海蓝色 `#0EA5E9`（印尼海洋）
- **成功色（Success）**：翠绿 `#22C55E`
- **背景（Background）**：纯白 `#FFFFFF` / 深色 `#0F172A`
- **卡片（Surface）**：浅灰 `#F8FAFC` / 深色 `#1E293B`
- **文字（Foreground）**：深炭 `#1E293B` / 浅色 `#F1F5F9`

## 屏幕列表

### 1. 首页（Home）
- 顶部：用户打卡连续天数（Streak）+ 今日目标进度
- 中部：学习地图（竖向滚动的关卡地图，类似 Duolingo）
- 每个单元显示为圆形图标节点，已完成为实色，未解锁为灰色
- 底部：底部导航栏

### 2. 课程列表页（Courses）
- 分为"入门（Pemula）"和"进阶（Menengah）"两个 Tab
- 每个单元显示为卡片：单元名称、进度条、课程数量
- 点击单元展开课程列表

### 3. 学习页面（Lesson）
- 顶部：进度条 + 体力值（Hearts）
- 内容区：词汇卡片（印尼语 + 中文 + 发音提示）
- 练习区：多选题 / 排列句子 / 填空题
- 底部：确认按钮

### 4. 测验结果页（Result）
- 大型庆祝动画（完成课程）
- 获得 XP 展示
- 错误词汇回顾
- 继续 / 返回主页按钮

### 5. 词汇闪卡（Flashcard）
- 翻转卡片动画（正面：印尼语，背面：中文+例句）
- 滑动手势：左（不会）/ 右（会了）
- 进度指示器

### 6. 学习统计页（Stats）
- 今日学习时间
- 本周学习天数（日历热图）
- 词汇掌握数量
- 成就徽章展示

### 7. 个人设置页（Profile）
- 学习目标设置（每日 XP 目标）
- 通知提醒设置
- 深色/浅色模式切换
- 关于应用

## 关键用户流程

**主线学习流程：**
首页地图 → 点击课程节点 → 学习页面（词汇讲解）→ 练习题目 → 结果页（获得 XP）→ 返回首页（节点变为完成状态）

**复习流程：**
复习 Tab → 词汇闪卡 → 滑动分类 → 完成复习

**打卡流程：**
每日首次打开 → 首页显示打卡提醒 → 完成一课 → Streak +1 → 动画庆祝

## 字体与图标
- 字体：系统默认（SF Pro on iOS, Roboto on Android）
- 图标：@expo/vector-icons MaterialIcons
- 印尼语文字：使用系统字体，确保拉丁字母清晰显示

## 动画规范
- 页面切换：水平滑动（Expo Router 默认）
- 按钮反馈：scale 0.97，duration 80ms
- 完成动画：confetti 效果 + 缩放弹出
- 卡片翻转：3D flip，duration 300ms
